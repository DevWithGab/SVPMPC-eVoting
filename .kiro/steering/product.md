# Product Overview

## SVMPC Voting System

A full-stack MERN application for managing cooperative elections and voting processes.

### Core Features
- **User Authentication**: Role-based access (admin, officer, member)
- **Election Management**: Create and manage elections with candidates and positions
- **Voting System**: Secure voting interface with results tracking
- **Admin Dashboard**: Manage users, elections, candidates, and generate reports
- **Staff Portal**: Officer-level management and activity monitoring
- **Announcements & Rules**: Communicate election information to members
- **Accessibility**: High contrast mode, adjustable font sizes, multi-language support (English, Filipino)
- **Activity Logging**: Track user actions and system events
- **PDF Generation**: Export voting results and reports

### User Roles
- **Admin**: Full system control, user management, election setup
- **Officer**: Election management, staff operations, activity monitoring
- **Member**: Vote in elections, view results, access resources

### Key Pages
- Landing, Login, Voting, Results, Elections, Candidates, Admin, Staff, Profile, Resources, Rules, Announcements
